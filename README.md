# MSP Pricing Calculator Directory

A lightweight public directory for free MSP and managed IT pricing calculators.

## Main calculator

[MSP Pricing Calculator](https://msp-pricing-calculator-vercel.vercel.app/) estimates managed IT services pricing from supported users, managed devices, servers, locations, SLA, security add-ons, support labor, tool cost, setup fee, and target gross margin.

Typical planning ranges are estimate-only, not vendor quotes. Many small business managed IT plans are modeled around roughly $100-$300 per user per month, while per-user MSP packages often compare around $100-$250 per user per month after scope, devices, security, SLA, and compliance work are normalized.

## Pricing models covered

- Per-user MSP pricing for office teams with similar support needs.
- Per-device pricing for shared workstations, kiosks, clinics, labs, and device-heavy environments.
- Tiered managed IT services pricing for Good / Better / Best package comparison.
- Co-managed IT pricing for internal IT teams that need MSP overflow, security, backup, or after-hours support.
- Security and compliance add-ons for regulated or risk-sensitive businesses.
