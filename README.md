# MSP Pricing Calculator Directory

A lightweight public directory for free MSP and managed IT pricing calculators.

## Main calculator

[MSP Pricing Calculator](https://msp-pricing-calculator-vercel.vercel.app/) estimates managed IT services pricing from supported users, managed devices, servers, locations, SLA, security add-ons, support labor, tool cost, setup fee, and target gross margin.
